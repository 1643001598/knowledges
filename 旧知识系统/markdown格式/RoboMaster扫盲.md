      - 比赛时刻
        - def
          - 全文所提及的“X 分钟 Y 秒”，即倒计时为（6-X）:（59-Y）；“X 分钟 Y 秒至 M分钟 N 秒期间”，即倒计时为（6-X）:（59-Y）至（7-M）:（60-N）

      - 总剩余血量
        - 每局比赛结束，己方所有存活机器人剩余血量的总值。
  - 阵容
    - 机器人阵容（最低4台）
      - 英雄（1）
        - 弹丸
          - 42mm（唯一）
        - ftr
          - 初始区域
          - 运行方式
            - 最多一个遥控器和一个自定义控制器
          - 多机通信
            - 允许
          - 发射机构
          - 射击初速度上限
            - 12m/s（远程优先）
            - 16.5m/s（部署模式）
          - 扣血与超限惩罚机制
            - 射击初速度超限
            - 射击热量超限和冷却
            - 底盘功率超限
            - 攻击伤害或撞击
            - 裁判系统模块离线
            - 判罚伤害
          - 回血与复活
            - 回血方式
              - 占领补给区/远程兑换血量
            - 复活方式
              - 读条复活/兑换立即复活
          - 弹丸的装载和补给
            - 准备阶段，预装42mm与17mm弹丸
            - 初始允许发弹量：0
            - 比赛阶段，可通过补给区，基地增益点，前哨战增益点兑换，或远程兑换允许发弹量
          - 经验与性能体系
            - 适用
          - 可占领的增益点
            - 基地增益点
            - 中央高地增益点
            - 梯形高地增益点
            - 地形跨越增益点（公路，高地，飞坡，隧道）
            - 前哨战增益点
            - 补给区增益点
      - 工程（2）
        - 经验与性能体系
          - 初始/上限血量250，底盘上限120W，除此之外不适用
        - 回血
          - 占领补给区
      - 步兵（3/4）
        - 发射机构
          - 17mm发射机构
        - 初速度上限
          - 25m/s
        - 可占领的增益点
          - +堡垒增益点
      - 空中（6）
        - 初始区域
          - 停机坪
        - 运行方式
          - 最多2个遥控器
        - 弹丸的装载和补给
          - 允许发弹量
            - 750
          - 不允许额外兑换
      - 哨兵（7）
        - 运行方式
          - 最多只有一个遥控器用于调试
        - 多机通信
          - 可以向所有己方机器人发送数据，但只能接收雷达的数据
        - 经验与性能体系
          - 不适用
          - 性能与机器人操作方式相关
            - 自动哨兵机器人
              - 血量上限400
              - 射击热度上限260
              - 射击热量冷却速率30/s
              - 底盘功率上限100W
            - 半自动哨兵机器人
              - 血量上限200
              - 射击热量上限100
              - 射击热量冷却速率10/s
              - 底盘功率上限60W
        - 增益点
          - +堡垒增益点
      - 飞镖系统（8）
        - 初始区域
          - 飞镖发射站
        - 运行方式
          - 最多仅1个遥控器
      - 雷达（9）
        - 初始区域
          - 雷达基座
        - 运行方式
          - 自动运行
        - 多机通信
          - 向所有己方机器人发送数据，只能接收哨兵机器人的数据
    - 操作手阵容
      - 英雄1
      - 工程1
      - 步兵2
      - 哨兵0
      - 云台1（包括空中机器人与飞镖系统）
      - 飞手1（空中机器人）
    - attention
      - 操作手只能由本赛季参赛队伍的正式队员和战队指导担任。
      - 每局比赛结束后，可以从本场场地人员中的正式队员和战队指导中替换操作手。
      - 无论空中机器人、飞镖系统是否上场，云台手都可上场操作空中机器人、飞镖系统对应的、裁判系统选手端，佩戴对应的耳机，三分钟准备阶段结束后不得移动位置。
      - 飞手需获得飞手资格，并佩戴飞手手环，才可在比赛中操控空中机器人。
