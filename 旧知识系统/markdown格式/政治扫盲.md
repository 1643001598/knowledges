- 3.传统
  - content
    - 崇尚精神
      - def
        - 侧重于精神
        - 它讲的是一种选择——在物质和精神之间，更看重精神成长和价值实现的一种文化倾向
        - 崇尚精神，就是中华文化不满足于仅仅“活着”或“活得好”，更追求“活得有意义、有境界、有格调”。
      - 表现
        - 对物质生活与精神生活相互关系独到的理解
          - 物质
            - 充实无忧
          - 精神
            - 道德境界充分升华
          - “仓廪实”不是终点，“知礼节”才是目标。 中国人认为，吃饱穿暖之后，人应该去追求更高尚的东西（比如修身养性、帮助他人、贡献社会），而不是一味地追求更贵的车、更大的房子。这种“不沉迷于物欲”的态度，就是“崇尚精神”。
        - 对理想的不懈追求
          - 民族精神内核
          - 中国人特别相信“未来会更好”，并且愿意为这个更好的未来去努力、甚至牺牲。
        - 对品格养成的重视
          - 人的性质
          - 评判一个人，不只是看他“有什么”（财富、地位），更要看他“是什么样的人”（诚信、仁爱、坚韧、担当）。 中国文化里有一整套做人的标准（如仁义礼智信），强调通过自我修养成为一个高尚的人。
      - 内涵
        - 创造精神
          - 人的天性/正向叙事的必然结果
        - 奋斗精神
          - 被包装的人的价值引领必将导向的结果/正向叙事的必然结果
        - 团结精神
          - 集体主义/正向叙事的必然结果
        - 梦想精神
          - 理想/引领性/正向叙事的必然结果
      - 在基本物质需求满足后，对精神世界的建设和对高尚价值的追求，才能带来更持久、更深层的幸福感和成就感。

- 4.价值
  - 社会主义核心价值观
    - ftr
      - 反映人类社会发展进步的价值理念
        - content
          - 体现社会主义的本质属性
            - 性质
            - 它姓“社”。像“共同富裕”（与“富强、平等”相关）、“人民当家作主”（与“民主”相关）这些核心理念，是资本主义价值观体系无法完全涵盖的。它标志着我们走的是另一条路。
          - 扎根中华优秀传统文化土壤
            - 特色社会主义/社会主义核心价值观结合了科学社会主义与中华精华文化的精神
            - 翻译：它有中国味儿。这不是外来价值观的生硬移植。

例子：

“爱国” -> “天下兴亡，匹夫有责”（顾炎武）

“敬业” -> “敬业乐群”（《礼记》）

“诚信” -> “人而无信，不知其可也”（《论语》）

“和谐” -> “和为贵”、“天人合一”
          - 吸纳世界文明有益成果
            - 博采众长，兼容并蓄
            - 它不排外、不自闭。像“民主、自由、平等、法治”这些词汇，本身也是人类共同追求的价值共识。我们承认其进步性，并将其纳入我们的体系，但注入了我们自己的理解和实践内涵。
        - core
          - 超越性
          - 精神图腾
          - 政治理想（社会主义）+ 文化根基（中华传统）+ 世界共识（人类文明成果）的三合一
      - 彰显人民至上的价值立场
        - content
          - 尊重人民群众历史主体地位
            - 群众史观（哲学观）
            - 历史不是英雄创造的，而是人民群众创造的
          - 体现人民为中心的价值导向
            - （实践观）
            - 翻译：所有价值的落脚点，必须是“人民的福祉”。比如：

“富强”是为了让人民过上好日子。

“民主”是为了让人民有参与感。

“公正”是为了保障人民的权利。

“友善”是为了构建和谐的人际关系。

它反复强调：国家、社会发展的最终目的是人本身。
        - core
          - 溯根性，本质
          - 它的根，扎在“人民”之中。它的本质是 “人民本位”。这既是政治宣言，也是一种道义承诺：我们所做的一切，终极目标是为了每一个具体的、活生生的人。
      - 因真实科学而具有强大的道义力量
        - content
          - 社会主义核心价值观是真实可信且科学的
            - 它不只是宣传口号，而是 “正在被实践和追求的目标” 。国家会通过法律、政策、教育、宣传等一切手段，努力让这些价值在社会生活中体现出来。比如：

通过《民法典》和司法改革追求“法治”和“公正”。

通过脱贫攻坚和乡村振兴追求“富强”和“平等”。

通过反腐倡廉追求“清正”和“文明”。

它宣称：我们不是在虚构一个乌托邦，而是在建设一个 “可实现的理想社会” 。这种“正在进行时”的状态，就是其“真实性”的论据。
            - 它的提出不是随意的，而是基于对 “社会发展规律”（马克思主义）和“中国现实国情” 的分析。它被认为是一套符合历史发展方向的、逻辑自洽的价值体系。
          - 认清西方”普世价值“的实质
