  - 第一节:方向决定道路，道路决定命运
    - 中特 是历史和人民的选择
      - 核心观点
        - 中特 不是凭空产生的，而是近代以来中国社会发展的 必然结果，是历史和人民的选择
      - 知识点
        - 历史背景
          - 1840鸦片战争后，中国逐步沦为半殖民地半封建社会，无数仁人志士探索救国道路，但都失败了（洋务运动，戊戌变法，辛亥革命）
          - 十月革命一声炮响，给中国带来了马列主义，中国先进分子看到了新希望
        - 中国共产党领导的作用
          - 中共应运而生，肩负起民族复兴的历史大任
          - 以毛泽东同志为主要代表的中共党人，把马列主义同中国具体实际相结合，提出：新民主主义革命走向社会主义 的正确理论
        - 社会主义建设的两个历史时期
          - 改革开放前：确立了社会主义基本制度，为后来的发展积累了条件
          - 改革开放后：在坚持前一个时期成果的基础上，进行改革，发展。
          - 重要结论：不能用改革开放后的历史时期否定改革开放前，反之亦然，两者是相互联系，不可分割的
        - 中国特色社会主义的“六个得来”
          - 内容
            - 是 在改革开放40多年的伟大实践中得来的
            - 是 在新中国成立70周年的持续探索中得来的
            - 是 在党领导人民进行伟大社会革命100多年的实践中得来的
            - 是 在近代以来中华民族由衰转盛180多年的历史进程中得来的
            - 是 在世界社会主义500多年波澜壮阔的发展历程中得来的
            - 是 在对中华文明5000多年的传承发展中得来的
          - 记忆方式：线索
            - 改革开放->新中国成立->党成立->中华民族由盛转衰->世界社会主义->中华文明5000年
