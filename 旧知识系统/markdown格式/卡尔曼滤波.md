# 卡尔曼滤波
- ABS
  - 核心思想
    - 预测：你有一个运动模型（比如它正以每秒10米的速度向北飞），所以你可以预测它下一秒应该在什么位置。
    - 测量：同时，你的GPS传感器会给你一个测量值，告诉你它现在的位置。但GPS有误差，可能不准确。
    - 融合：卡尔曼滤波的核心就是智能地融合你的“预测”和“测量”，得到一个比单独使用任何一个都更精确、更可靠的“最佳估计”。
  - definition
    - 简单来说，卡尔曼滤波是一个最优化递归数据处理算法。
    - 最优化：它给出的结果是统计意义上最优的（通常是均方误差最小）。
    - 递归：它不需要所有历史数据，只需要上一时刻的最优估计和当前时刻的测量值，就能计算出当前时刻的最优估计。这使得它计算量小，非常适合实时应用。
    - 数据处理：它处理的是带有噪声的数据。
  - core
    - 递归数据处理器
- 适用场景
  - 线性系统：系统的动态模型和测量模型都是线性的。（对于非线性系统，有扩展卡尔曼滤波-EKF和无迹卡尔曼滤波-UKF等变体）。
  - 高斯噪声：系统过程噪声（模型的不确定性）和测量噪声都服从高斯分布（正态分布）。
- 核心原理
  - KF
    - 基本元（变量定义）
      - 状态变量x相关
        - xk_hat
          - 在时刻k的状态估计向量
      - Pk
        - 在时刻k的估计协方差矩阵
      - Fk
        - 状态转移矩阵
可以理解为状态空间基于经验
    - 过程
      - 预测
        - 1. 状态预测
          - 解释
            - xk_hat^(-1)
              - 第k时刻的先验状态估计
            - F
              - 状态转移矩阵
            - xk-1_hat
              - k-1时刻的后验状态估计
            - B
              - 控制输入矩阵，它将控制量uk（如加速度）的影响映射到状态上（外部影响）
            - uk
              - 控制量（外部影响）
        - 2.协方差预测
          - 解释
            - Pk^(-1)
              - 先验估计协方差: 是一个双线性矩阵
            - Pk-1
              - 上一时刻的后验估计协方差
            - Q
              - 过程噪声协方差，表示我们的运动模型本身的不确定性
例如，目标可能突然加速或减速
Q越大，模型越不可靠
      - 更新
        - 3.卡尔曼增益计算
          - 解释
            - Kk(n x m)
              - 卡尔曼增益（core）
含义：在多大程度上相信“预测”/“测量”
把观测向量残差映射到状态向量的改变量
              - 内部组成
                - 前者（n x m）
                  - 状态与观测的交叉协方差
                  - 含义
                    - 把观测的残差 -> 状态空间的修正
                - 后者（m x m）
                  - 总的预期观测不确定性的逆
                  - 含义
                    - 加权（咋加权的就不知道了）
            - H（m x n）
              - 观测矩阵
左乘：将状态向量x(n维)映射到测量域(m维)
            - R（m x m）
              - 测量噪声协方差
                - 测试设备的精度
R越大，测量值越不可靠
        - 4.状态更新
          - 解释
            - xk_hat
              - k时刻后验状态估计
            - zk(m维向量)
              - 实际测量值
            - zk-H * xk_hat^(-1)
              - 残差（预测值与测量值之间的差异）
            - 等价于
              - xk_hat = xk_hat^(-1) * (1 - K_k * Hxk_hat^(-1))  +  K_k * z_k
        - 5.协方差更新
          - 解释
            - Pk
              - 更新后的后验估计协方差
    - 示例
      - title
        - 估算房间的温度
      - 基本元
        - 状态：温度T
      - 预测
        - 根据经验，我们相信房间温度是恒定的。所以我们的模型是：下一时刻温度 = 当前温度。即
F = 1
        - 但房间温度可能会有缓慢变化，所以我们设一个较小的过程噪声Q
      - 测量
        - 不太准确的温度计 -> 测量噪声协方差：R
      - 过程
        - 初始化
          - 随机初始化一个温度，给一个很大的初始不确定性P0
        - 迭代
          - 预测
            - 用上一时刻 的最优估计来预测当前温度，同时增加不确定性（加了Q）
          - 更新
            - 读取温度计测量值zk
            - 计算卡尔曼增益K
            - 将预测值与测量值按权重K融合，得到最优温度估计
            - 更新Pk
  - EKF
    - ftr
      - KF的非线性扩展版本
      - 局部线性化：通过雅可比矩阵来近似线性
    - （非线性系统）模型
      - 状态方程
        - x_k = f(x_{k-1}, u_k) + w_k
        - 参数解释
          - f：非线性函数
        - def
          - 描述状态如何演化
      - 观测方程
        - z_k = h(x_k) + v_k
        - 参数解释
          - h：使用当前的预测状态，得出理论上测量的结果的矩阵
          - v_k：高斯白噪声
        - def
          - 描述状态如何被测量
    - 过程
      - 预测
        - 状态预测
          - x̂_k|k-1 = f(x̂_{k-1|k-1}, u_k)
        - 协方差预测
          - P_k|k-1 = F_k * P_{k-1|k-1} * F_k^T + Q_k
          - 参数解释
            - F_k 是状态函数 f（状态变换矩阵F） 在 x̂_{k-1|k-1} 处的雅可比矩阵（即线性化后的状态转移矩阵）。
            - Q_k 是过程噪声的协方差矩阵。
            - 状态函数f
              - 它是一个有n个函数分量构成的向量，每个函数输入的都有n个，是上一个时刻的状态，每个位置上函数的输出的含义就是：基于n个对其有影响的因素，影响之后，得到的结果
      - 更新
        - 计算卡尔曼增益
          - K_k = P_k|k-1 * H_k^T * (H_k * P_k|k-1 * H_k^T + R_k)^{-1}
          - 参数解释
            - H_k 是观测函数 h（观测矩阵H） 在 x̂_k|k-1 处的雅可比矩阵（即线性化后的观测矩阵）。
            - R_k 是观测噪声的协方差矩阵。
        - 状态更新
          - x̂_k|k = x̂_k|k-1 + K_k * (z_k - h(x̂_k|k-1))
          - 参数解释
            - (z_k - h(x̂_k|k-1)) 被称为新息，是实际观测值与预测观测值之间的差异。
        - 协方差更新
          - P_k|k = (I - K_k * H_k) * P_k|k-1
          - 含义
            - 更新状态估计的不确定性
